Challenge 2: Order Confirmation Modal

Objective: Replace the browser alert() with a custom modal when an order is created.

Requirements:

  • Modal should appear with a smooth fade-in animation
  • Show success icon (checkmark) at the top
  • Display order ID in a highlighted box
  • Include a "View Order" button and "Continue Shopping" button
  • Modal should have a semi-transparent overlay
  • Clicking outside the modal or "Continue Shopping" should close it
  • Modal should be responsive and centered
